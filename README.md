# Training for Microsoft Course PL-7008: Create agents in Microsoft Copilot Studio

Companion Material and Resources for Class Delivery by [**Ricardo Cabral**](https://www.rramoscabral.com) for Microsoft Course PL-7008: Create agents in Microsoft Copilot Studio.

<br/>

The course *PL-7008: Create agents in Microsoft Copilot Studio* is intended for functional consultants and makers who want to build and deploy custom conversational agents (bots) using Microsoft Copilot Studio.

<br/>

**Learning objectives**

After completing this course, students will be able to:
- Get started with Microsoft Copilot Studio
- Build an initial agent with Microsoft Copilot Studio
- Manage topics in Microsoft Copilot Studio
- Work with entities and variables in Copilot Studio
- Enhance agents with Copilot Studio integrations
- Create agents with Microsoft Copilot Studio and Microsoft Dataverse for Teams
