## Applied Skills credential

This course prepares for the Microsoft Applied Skills credential: **Create agents in Microsoft Copilot Studio**.

**Applied Skills credentials are not replacing Certifications.**

<br/>

## Skills measured

The Applied Skills assessment evaluates your ability to:
- Get started with Microsoft Copilot Studio
- Build an initial agent
- Manage topics and conversation flows
- Work with entities and variables
- Enhance agents with integrations (Power Automate, Omnichannel)
- Create agents with Dataverse for Teams

<br/>

## Microsoft Applied Skills

Microsoft Applied Skills is a verifiable credential that validates that you have the targeted skills needed to implement critical projects aligned to business goals and objectives. **Applied Skills credentials are not replacing Certifications.**

<br/>

## How to earn your Microsoft Applied Skills credential

Earn your credential by passing an online, interactive, lab-based assessment that takes you through a series of scenario-based tasks in Microsoft Copilot Studio. The lab assessment is available directly from Microsoft Learn.

<br/>

| Applied Skills | Overview |
| --- | --- |
