## Course PL-7008

<!-- Overview -->
This course covers how to build and deploy custom conversational agents (bots) using Microsoft Copilot Studio. Students will learn to create agents from scratch, manage topics and conversation flows, work with entities and variables, integrate with Power Automate, and leverage Microsoft Dataverse for Teams.

> NOTE: To complete the interactive labs in this course, learners need access to a **Microsoft 365 environment** with **Microsoft Copilot Studio** licensing or trial.


| Level        | Product                   | Role                              | Language | Course Duration |
| ---          | ---                       | ---                               | ---      | --- |
| Intermediate | Microsoft Copilot Studio  | Functional Consultant / Maker     | English  | 1 day |
