## Official course Hands-on Labs

* ***PL-7008*** official course lab instructions are available [**here**](https://microsoftlearning.github.io/mslearn-copilotstudio/).
* You can download the latest instructions and complete solution [**here**](https://github.com/MicrosoftLearning/mslearn-copilotstudio).

> ***Note:*** Microsoft Learning has the right to remove the repository containing the hands-on labs. When a course is set to be retired for any reason the repository supporting hands-on labs is removed by Microsoft Learning. You can consult the Course retirement [here](https://learn.microsoft.com/en-us/credentials/certifications/retired-courses).


> NOTE: To complete the interactive labs in this course, learners need access to a **Microsoft 365 environment** with **Microsoft Copilot Studio** licensing or a trial environment.
