## What is Microsoft Copilot Studio?

Microsoft Copilot Studio is a low-code platform that enables you to build custom conversational agents (bots) for your business. It was formerly known as Power Virtual Agents. You can create agents that can answer questions, automate processes, and integrate with your business data and services.

## What licenses are required to use Microsoft Copilot Studio?

You need a Microsoft Copilot Studio license or a Power Virtual Agents license to use the full feature set. Copilot Studio is also included in some Microsoft 365 and Dynamics 365 subscriptions. For detailed licensing information, see the [Licensing & Pricing](./licensing-pricing.md) section.

## Can I try Microsoft Copilot Studio for free?

Yes, Microsoft offers a trial environment for Copilot Studio. You can sign up at [https://copilotstudio.microsoft.com](https://copilotstudio.microsoft.com) and start building agents without a paid subscription during the trial period.

## What is the difference between an agent and a bot?

In Microsoft Copilot Studio, the terms "agent" and "bot" are used interchangeably. Microsoft has updated its terminology to use "agent" instead of "bot" to better reflect the AI capabilities of the platform.

## What is Dataverse for Teams?

Dataverse for Teams is a built-in, low-code data platform that comes with Microsoft Teams. It provides relational data storage, rich data types, enterprise-grade governance, and one-click solution deployment to power Teams applications, including agents built with Copilot Studio.
